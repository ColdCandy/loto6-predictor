【公開サイト（ログイン不要・予想のみ）】
  https://coldcandy.github.io/loto6-predictor/
  ・PCオフでもOK
  ・最新当選・AI本命の自動更新あり

【高機能版（招待制・パスワード必須）】
  1. このPCで「招待パスワード設定.bat」を実行
  2. 家族ごとの招待ID / パスワードを決めて保存
  3. 「予想GUI.bat」または「外から起動.bat」で起動
  4. 相手には「URL + 招待ID + パスワード」を個別に伝える

  Streamlit Cloud を使う場合:
  - Sharing は Public でも中身はログイン必須
  - 同じ secrets 内容を Cloud の Secrets に貼る
  - Main file: streamlit_app.py

【注意】
  ・secrets.toml は Git に上がりません
  ・パスワードはSNSに書かない
  ・完璧なゼロリスクではありませんが、招待制で大幅に安全です
